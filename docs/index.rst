Occitanie MaaS Scenario Generator documentation
===============================================

`eqasim-france <https://github.com/eqasim-org/eqasim-france>`_ is an open pipeline for generating synthetic populations and mobility demand in France.
This repository documents the current project adaptation focused on the Occitanie MaaS scenario workflow (baseline-driven generation, latent classes, and scenario export).
